### 1. Source-First Development
To maintain full compliance with the copyleft spirit of the LGPLv3, all source code for this mod is developed openly and published to this repository prior to, or simultaneously with, the compilation of any functional artifacts (the `.jar` files). The public availability of this source code satisfies our obligation to provide the Corresponding Source to all users.

### 2. Early-Access & Beta Testing Binaries
We utilize a source-first, binary-delayed release model for quality assurance and project sustainability:
* **The Source Code:** Always open, public, and free to review, fork, or compile yourself under the terms of the LGPLv3.
* **Pre-Compiled Artifacts:** Beta builds and experimental testing binaries may be distributed privately to specific testing groups (such as project patrons) for bug-reporting and stability testing before being released to public mod distribution platforms (like CurseForge and Modrinth).

Receiving a pre-compiled binary as part of an early testing tier does not alter your rights under the LGPLv3 regarding the source code available in this repository.
